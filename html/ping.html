Small errors that can make the ping tool not to work properly

From the examples below, it can be seen how some silly things can give undesirable result.

Two simple thing that can mess up the ping tool are:

    - just by adding https:// in the URL

    eg. Https://abc.com

    - or just the single back slash in a URL

    eg. abc.com/

Therefore the only format it accepts is:

    - clean URL

    eg. abc.com

    - ip address

    eg. 192.168.43.1
